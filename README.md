# link-fh
一个简单通用的cos防洪系统

#部署文档
1.环境要求，Nginx 1.26.3 PHP8，MySQL 5.7.44

2.将代码上传到你站点后解压：【配置数据库：config.php】随后导入到数据库即可

3.后台就简单一个功能，添加你的防红链接到数据库中调用，
